🔥 热沙箱池

预热复用沙箱省冷启动；按语言分桶、自动健康检查 + TTL/空闲回收。开启后 Worker 派发任务时从池借用。

⚠️ 开关即时写入并启停后台回收器；已在运行的 Worker 在下次任务派发时生效。池参数（max_total / TTL 等）改动需重启 API。

📢 通知渠道 系统通知推送到外部

系统每产生一条通知（任务完成/失败/待审等）即推送到下方 enabled 的渠道。每行下拉选渠道类型（飞书/钉钉/企业微信/Slack/通用），填 Webhook URL。事件留空=接收全部。改动后需点「保存渠道」。
